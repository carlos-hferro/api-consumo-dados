# Consumo de API

Projeto desenvolvido para estudos de consumo de API, manipulação de dados e integração entre frontend e serviços externos.

## Tecnologias utilizadas

- JavaScript
- React
- API REST
- HTML/CSS

## Funcionalidades

- Consumo de API
- Manipulação de dados
- Exibição dinâmica de informações
- Requisições HTTP

## Objetivo

Projeto acadêmico desenvolvido para aprimorar conhecimentos em integração com APIs e desenvolvimento frontend.
